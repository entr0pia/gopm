# Go Package Manager

Search go packages from [pkg.go.dev](https://pkg.go.dev/). And more in the feature.

## Usage

```
gopm search [-m] -n package_name
  -m  more packages
  -n string
      package name
```

## Installation

```shell
go get -u github.com/entr0pia/gopm
go install github.com/entr0pia/gopm@latest
```
